## 📌 About Project

This project is a **basic HTML email template** created for promotional campaigns like sales and offers.  
It is designed using **table-based layout and inline CSS** to ensure compatibility with different email clients.

---

## 🚀 Features

- Mobile-friendly design  
- Table-based layout  
- Inline CSS styling  
- Call-to-action button  
- Simple and clean UI
